📱 Wallpaper App
A native Android application built from scratch in Android Studio using Java, that lets users browse wallpapers and set them directly as their device wallpaper.

✨ Features
🖼️ Browse a collection of wallpapers in a clean grid layout
🔍 Full-screen preview of a wallpaper before applying it
⚡ One-tap "Set as Wallpaper" using Android's WallpaperManager API



🛠️ Built With:
Java — core application logic
Android SDK — native Android development
RecyclerView & GridLayoutManager — for the wallpaper grid
WallpaperManager API — to apply the selected wallpaper
Android Studio — IDE
